# Практическое занятие №18: Lighthouse и доступность

## Описание проекта
Этот проект содержит исправленную версию страницы контактов с улучшенной доступностью и отчеты Lighthouse для 5 веб-страниц.

## Структура файлов
- `index.html` - исправленная доступная версия страницы контактов
- `original-index.html` - исходная версия с проблемами доступности
- `css/style.css` - стили для доступной версии
- `js/script.js` - JavaScript для улучшения доступности

## Особенности реализации
- **Адаптивность**: Оптимизация для мобильных устройств
- **Производительность**: Оптимизированные стили и скрипты

## Ключевые улучшения доступности
- Семантическая HTML-разметка
- Правильные ARIA-атрибуты
- Достаточная контрастность цветов
- Навигация с клавиатуры
- Подписи для всех элементов форм
- Управление фокусом
- Сообщения об ошибках для скринридеров
- Skip-links для быстрой навигации

## Быстрые клавиши
- `Alt + 1` - переход к заголовку
- `Alt + 2` - переход к форме
- `Tab` - циклическая навигация по форме

## Запуск
Откройте файл `index.html` в браузере для просмотра доступной версии страницы.

## Тестирование
Для проверки доступности используйте:
- Lighthouse в DevTools
- Клавиатурную навигацию
